## Version 0.7.x (2021)

### 0.7.1 ‚Äî 2021-03-05

- Bumped `lodash` dependency to address a security advisory.

### 0.7.0 ‚Äî 2021-03-03

- **Security fix:** Patched a code injection vulnerability when processing string literals.
- Fixed `GROUP BY` returning no rows instead of an empty result set when the input is empty.

## Version 0.6.x (2020)

### 0.6.5 ‚Äî 2020-11-11

- Fixed `COUNT` and `AVG` incorrectly including `null` values.

### 0.6.4 ‚Äî 2020-09-24

- Added support for `String` and `Number` objects as column values.
- `JOIN USING` now handles `String` and `Number` objects correctly.
- Fixed incorrect file naming when exporting to Excel.

### 0.6.2 ‚Äî 2020-05-31

- `NOT BETWEEN` can now be written without parentheses.

### 0.6.1 ‚Äî 2020-05-20

- `BETWEEN` can now be written without parentheses.
- `ADD COLUMN` now respects the specified `dbtypeid`.

### 0.6.0 ‚Äî 2020-05-02

- Added `NULLS FIRST` / `NULLS LAST` clause to `ORDER BY`.
- Table and row names can now start with a number.

## Version 0.4.x (2017‚Äì2018)

### 0.4.11‚Äì0.4.10 ‚Äî 2018-10

- Added ability to specify an XLSX sheet without knowing its name.
- Added support for XLSXML files with multiple sheets.

### 0.4.5 ‚Äî 2018-01-24

- Fixed aggregate functions applied to empty sets.
- Fixed `ORDER BY` direction in AST `toString()`.
- Fixed `toString()` on multi-join syntax trees.

### 0.4.2 ‚Äî 2017-08-17

- Added `LTRIM` and `RTRIM` string functions.
- Out-of-the-box Webpack and Browserify compatibility without extra configuration.

### 0.4.0 ‚Äî 2017-05-09

- **Breaking:** `OFFSET N` now skips the first N rows (previously skipped N‚àí1).
- Added quartile aggregators: `QUART`, `QUART2`, `QUART3`.
- Fixed `MEDIAN` and `STDEV` aggregators with `ROLLUP`.
- Fixed `SHOW COLUMNS` and `SHOW INDEX` with the promise interface.

## Version 0.3.x (2016‚Äì2017)

### 0.3.9 ‚Äî 2017-03-23

- Added React Native support.

### 0.3.8 ‚Äî 2017-03-15

## Version 0.2.x (2015‚Äì2016)

### 0.2.7 ‚Äî 2016-05-30

- Added Node.js 6.0 support.
- `.promise()` now returns all responses, not just the last one.
- CSV/Excel `INTO` and `FROM` statements default headers to `true`.

### 0.2.3 ‚Äî 2016-02-01

- Added user-defined aggregator functions.
- New fast aggregator calculation engine.
- Added `REPLACE()`, `NEWID()` / `UUID()` / `GEN_RANDOM_UUID()` functions.
- `^` operator is now XOR; `~` is binary NOT.

### 0.2.2 ‚Äî 2016-01-15

- Added `PIVOT` and `UNPIVOT`.
- Added `REPLACE INTO`.
- Added triggers.
- Added `DATEADD()` and `DATEDIFF()` (T-SQL style).
- Added binary operators: `>>`, `<<`, `&`, `|`, `||` (string concatenation).

### 0.2.1 ‚Äî 2015-09-28

- Added `REGEXP` operator and `REGEXP_LIKE()` function.
- Added `INSERT OR REPLACE`.
- Added `LIKE ESCAPE`.
- CLI now reports a proper exit code.

### 0.2.0 ‚Äî 2015-07-13

- Documentation overhaul and code refactoring release.
