AlaSQL is an open source SQL database for JavaScript with a strong focus on query speed and data source flexibility. It works in the web browser, Node.js, and mobile apps, letting you use familiar SQL syntax on your JavaScript data, spreadsheets, and local storage.

## Quick example

```javascript
import alasql from 'alasql';

// Query an array of objects directly
const data = [
  { city: 'Paris', population: 2249975 },
  { city: 'Berlin', population: 3517424 },
  { city: 'Madrid', population: 3041579 },
];

const result = alasql('SELECT * FROM ? WHERE population < 3500000 ORDER BY population DESC', [data]);
// [{ city: 'Madrid', population: 3041579 }, { city: 'Paris', population: 2249975 }]
```

<Note>
  AlaSQL works on both relational (tabular) and nested JSON data. No schema setup required — pass your JavaScript arrays directly as query parameters.
</Note>
