<Steps>
  <Step title="Install">
    Add AlaSQL to your project with your package manager of choice.

    <CodeGroup>

    ```bash npm
    npm install alasql
    ```

    ```bash yarn
    yarn add alasql
    ```

    ```bash pnpm
    pnpm add alasql
    ```

    </CodeGroup>

    <Note>
      AlaSQL requires Node.js 15 or later. For browser usage, see the [CDN option](/installation#cdn) on the installation page.
    </Note>
  </Step>

  <Step title="Import">
    Import AlaSQL into your file using ESM or CommonJS syntax.

    <CodeGroup>

    ```javascript ESM
    import alasql from 'alasql';
    ```

    ```javascript CommonJS
    const alasql = require('alasql');
    ```

    </CodeGroup>

    AlaSQL exports a single callable function. In the browser you can also include the script tag directly and use `alasql` as a global — see [Installation](/installation).
  </Step>

  <Step title="Run a query on a JavaScript array">
    Pass your data as a `?` parameter and write a standard SQL `SELECT` against it. No schema or table creation required.

    ```javascript
    import alasql from 'alasql';

    const products = [
      { id: 1, name: 'Widget',  category: 'tools',  price: 9.99  },
      { id: 2, name: 'Gadget',  category: 'tools',  price: 24.99 },
      { id: 3, name: 'Gizmo',   category: 'parts',  price: 4.99  },
      { id: 4, name: 'Doodad',  category: 'parts',  price: 14.99 },
    ];

    const result = alasql(
      'SELECT category, COUNT(*) AS count, SUM(price) AS total FROM ? GROUP BY category',
      [products]
    );

    console.log(result);
    // [
    //   { category: 'tools', count: 2, total: 34.98 },
    //   { category: 'parts', count: 2, total: 19.98 },
    // ]
    ```
  </Step>

  <Step title="Use results">
    `alasql()` returns a plain JavaScript array of objects. You can filter, map, or pass it directly to a rendering library.

    ```javascript
    const cheapest = result.sort((a, b) => a.total - b.total)[0];
    console.log(`Cheapest category: ${cheapest.category} ($${cheapest.total.toFixed(2)})`);
    // Cheapest category: parts ($19.98)
    ```
  </Step>
</Steps>

## Create and query in-memory tables

For repeated queries or when you want to keep data across multiple operations, create a named table:

```javascript
import alasql from 'alasql';

// Create the table
alasql('CREATE TABLE cities (city STRING, population NUMBER)');

// Insert rows
alasql("INSERT INTO cities VALUES ('Paris', 2249975), ('Berlin', 3517424), ('Madrid', 3041579)");

// Or load data directly into the table's data array for bulk inserts
alasql.tables.cities.data.push({ city: 'Rome', population: 2863223 });

// Query
const large = alasql('SELECT * FROM cities WHERE population > 3000000 ORDER BY population DESC');
console.log(large);
// [{ city: 'Berlin', population: 3517424 }, { city: 'Madrid', population: 3041579 }]
```

## Browser usage

Include the minified browser build from a CDN, then use `alasql` as a global:

```html
<!DOCTYPE html>
<html>
  <head>
    <script src="https://cdn.jsdelivr.net/npm/alasql@4"></script>
  </head>
  <body>
    <script>
      var data = [{ x: 1 }, { x: 2 }, { x: 3 }];
      var result = alasql('SELECT x * 2 AS doubled FROM ?', [data]);
      console.log(result); // [{ doubled: 2 }, { doubled: 4 }, { doubled: 6 }]
    </script>
  </body>
</html>
```

## Async queries and Promises

File I/O operations (Excel, CSV, SQLite) and multi-statement arrays are always asynchronous. Wrap the SQL in an array to get a Promise back, or use `alasql.promise()`:

```javascript
import alasql from 'alasql';

// Array syntax returns a Promise
alasql(['SELECT * FROM XLS("./data/sales.xls") WHERE region = "EMEA"'])
  .then((results) => {
    console.log(results[0]); // first result set
  })
  .catch(console.error);

// alasql.promise() wraps a single statement in a Promise
alasql
  .promise('SELECT * FROM CSV("./data/cities.csv", {headers: true})')
  .then((rows) => console.log(rows))
  .catch(console.error);
```

<Tip>
  Use `async`/`await` with the array syntax for cleaner code:

  ```javascript
  const [rows] = await alasql(['SELECT * FROM CSV("cities.csv", {headers: true})']);
  console.log(rows);
  ```
</Tip>
